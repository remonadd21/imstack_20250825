2. 글자 스타일

1)글자색상

2)글자크기

3)정렬
text-align은 글자들을 정렬할 때 사용하는 효과로, 속성으로는 center(가운데 정렬) / left (왼쪽으로 정렬) / right (오른쪽으로 정렬) / justify(수평정렬)이 있다.
4)밑줄
text-decoration:none; 깃 허브
5)글꼴
font-family:arial;
6)굵기
font-weight:bold;
